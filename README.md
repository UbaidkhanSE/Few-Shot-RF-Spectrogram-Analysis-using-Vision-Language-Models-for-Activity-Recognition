# Few-Shot-RF-Spectrogram-Analysis-using-Vision-Language-Models-for-Activity-Recognition
Few-Shot RF Spectrogram Analysis using Vision-Language Models for Activity Recognition:
Radio frequency (RF) spectrogram analysis for indoor human activity recognition has traditionally relied on supervised learning with larger labeled datasets. We present a novel framework that employs vision language models (VLMs), notably contrastive language-image pre-training (CLIP), achiev ing comparable accuracy from RF spectrograms with a few shot learning approach. Our framework provides domain-specific prompt engineering and an improved contrastive learning ap proach that effectively bridges the gap between natural language description and RF signal features. The system allows instan taneous spectrogram analysis using a user-friendly interface that allows classification of movement patterns, activity type, identification of subject presence, and a precise description. Our experimental results show promising performance in multiple scenarios, using only five base training samples with data augmentation required to achieve 85.5% accuracy in subject detection, 82.3% accuracy in movement pattern and 84.2% accuracy in activity type classification. We present a few-shot learning framework using VLMs that achieves these results with minimal labeled data, reducing implementation complexity while maintaining reasonable performance metrics.
